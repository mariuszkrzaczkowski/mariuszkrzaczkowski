## 👋 Who I am

I'm **Mariusz Krzaczkowski** — **Solutions Architect & Senior IT Engineer at [YetiForce](https://github.com/YetiForceCompany)**, an open-source enterprise CRM/ERP used by companies across multiple markets. For **15+ years** I've been building, scaling, and securing complex software systems end-to-end.

I'm a **full-scope engineer**: I think it's normal to design a multi-tenant data model on Monday, debug a low-level PHP extension on Tuesday, harden a Linux server on Wednesday, ship a CI/CD pipeline on Thursday, and review a vulnerability report on Friday.

My expertise spans across full development lifecycle—from architectural decisions and backend optimization to DevOps automation and security-first practices. I'm passionate about solving complex technical challenges and contributing to the open-source ecosystem.
